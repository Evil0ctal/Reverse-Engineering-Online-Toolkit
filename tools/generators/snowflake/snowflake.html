关于 Snowflake ID

Snowflake ID 是 Twitter 开发的分布式唯一 ID 生成算法。64 位整数分为：1 位符号位（始终为 0）、 41 位时间戳、5 位数据中心 ID、5 位 Worker ID、12 位序列号。
