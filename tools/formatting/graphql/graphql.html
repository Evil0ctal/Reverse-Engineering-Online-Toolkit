关于 GraphQL

                

                    
GraphQL 是一种用于 API 的查询语言，由 Facebook 开发。它提供了一种更高效、更强大的数据获取方式。


                    

                        

                            
操作类型

                            
    
                                
  • query - 查询数据（只读）
    • 
                                
  • mutation - 修改数据
    • 
                                
  • subscription - 实时订阅
    • 
                            

                        

                        

                            
主要特性

                            
    
                                
  • 按需获取数据字段
    • 
                                
  • 单次请求获取多个资源
    • 
                                
  • 强类型系统
    • 
                                
  • 自文档化 API
    • 
                            

                        

                        

                            
FB/IG API 支持

                            
    
                                
  • 自动检测 URL 编码请求
    • 
                                
  • 解析 doc_id 和 variables
    • 
                                
  • 提取 API 名称和参数
    • 
                                
  • 格式化 JSON 变量
    • 
                            

                        

                    


                    

                        
                        使用提示：支持标准 GraphQL 语法格式化，以及 Facebook/Instagram API 的 URL 编码请求解析。
