关于 EXIF

EXIF (Exchangeable Image File Format) 是一种图片元数据标准，用于存储拍摄参数、相机信息、GPS 位置等数据。

支持格式

  • JPEG/JPG - 最常见的 EXIF 载体
  • TIFF - 原生 EXIF 格式
  • HEIC/HEIF - Apple 设备格式
  • PNG - 有限 EXIF 支持
  • WebP/AVIF - 现代图片格式

数据分类

  • 基本信息 - 尺寸、时间、方向
  • 相机信息 - 厂商、型号、曝光参数
  • GPS 信息 - 经纬度、海拔、时间戳
隐私提示：EXIF 数据可能包含敏感的位置信息，分享图片前请注意检查。
