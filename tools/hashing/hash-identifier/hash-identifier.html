哈希类型参考

哈希类型 长度 格式特征 常见用途
MD5 32 字符 十六进制 文件校验、旧密码存储
SHA-1 40 字符 十六进制 Git 提交、旧证书
SHA-256 64 字符 十六进制 比特币、SSL 证书
SHA-384 96 字符 十六进制 TLS 证书
SHA-512 128 字符 十六进制 安全存储、签名
bcrypt 60 字符 $2a$/$2b$/$2y$ 前缀 密码存储
Argon2 可变 $argon2i$/$argon2d$/$argon2id$ 前缀 现代密码存储
NTLM 32 字符 十六进制 Windows 密码
MySQL 5.x 40 字符 * 前缀 + 十六进制 MySQL 密码
CRC32 8 字符 十六进制 文件校验
